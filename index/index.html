ping Pong en Java Script por Alvaro Ducuara

Debes presionar la tecla espacio para iniciar el juego

Como jugar al Ping Pong:

---- La barra izquierda es del jugador #1, presiona (w)-> subir barra, (s)-> bajar barra.

---- La barra derecha es del jugador #2, presiona flecha arriba -> subir barra, flecha abajo -> bajar barra

---- Aquel jugador quien envie la pelota fuera de la mesa, pierde.
